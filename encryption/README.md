Выполнена задача преобразования данных, чтобы скрыть персональную информацию о клиентах.

Использование алгоритма преобразования (умножение матрицы признаков на случайную обратимую матрицу) было обосновано математически, после чего было проведено соответствующее преобразование с нашими данными.

Измерение метрики R2 для моделей линейной регрессии, построенных на исходных и преобразованных данных, показало, что качество модели после преобразования данных не меняется.
